Frequently Asked Questions (FAQ)

For review/trial purposes only, please set the test server address to http://36.137.183.40:30020 on the login page, then use the provided test account and password to log in.

What should I do if login fails?

A: Please verify the following:

  • Server address is correctly configured (if using test environment)
  • Account and password are entered correctly
  • Network connection is stable
If the issue persists, contact our support team for troubleshooting.

Why is data synchronization abnormal?

A: Please check your network connection first. If the problem continues, contact support to verify server configuration and data sync status.
